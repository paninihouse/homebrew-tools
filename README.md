# Paninihouse Tools

## How do I install these formulae?

`brew install paninihouse/tools/<formula>`

Or `brew tap paninihouse/tools` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "paninihouse/tools"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
